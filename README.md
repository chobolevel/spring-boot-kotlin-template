# Spring Boot + Kotlin Backend Template

개인 백엔드 프로젝트를 빠르게 시작하기 위한  
Spring Boot + Kotlin 기반 템플릿입니다.

단순한 보일러플레이트가 아니라,  
확장 가능하고 구조적으로 명확한 백엔드 아키텍처를 목표로 합니다.

---

## 🛠 Tech Stack

- Spring Boot 3.x
- Kotlin (JVM)
- Gradle (Kotlin DSL)
- Spring Data JPA
- Spring Actuator
- JUnit 5
- Testcontainers (optional)
